# 🌐 Aononto Jahan — Personal Portfolio

> A modern, dark-themed personal portfolio website built with **HTML**, **CSS**, and **JavaScript** — designed to showcase skills, projects, education, and services as a Full Stack Developer & Software Engineer.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Vercel-black?style=for-the-badge&logo=vercel)](https://aonontojahan.vercel.app)

## ✨ Features

- 🎨 **Dark-themed UI** with smooth glassmorphism design
- 🌙 **Light / Dark Mode Toggle** with persistent preference
- 📱 **Fully Responsive** — works flawlessly across all device sizes
- 🧩 **Interactive Modal Cards** — click any card for detailed pop-up info
- ⚡ **Scroll-reveal Animations** for a dynamic, engaging experience
- 🔗 **Smooth Scroll Navigation** with active link highlighting
- 📥 **One-click CV Download** button
- 🔒 **Scroll-aware Navbar** that adapts on scroll

## 🚀 Getting Started

No build step required — this is a static site.

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aonontojahan/My-Portfolio.git
   cd My-Portfolio
   ```

2. **Open in browser:**
   Simply open `index.html` in your browser, or use a live-server extension in VSCode.

   ```bash
   # With VS Code Live Server
   code . # then right-click index.html → Open with Live Server
   ```

## ☁️ Deployment

This project is deployed on **Vercel** with zero configuration needed.

To deploy your own fork:

1. Push to GitHub
2. Import your repo at [vercel.com](https://vercel.com)
3. Vercel auto-detects static files — no build command needed

> **Note:** File paths are **case-sensitive** on Vercel's Linux environment. Ensure all asset directories (`images/`, `logos/`, `icons/`, `css/`, `js/`, `cv/`) are lowercase.
